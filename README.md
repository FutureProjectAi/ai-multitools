# AI Multitools Starter (Minimal)

This is a minimal Next.js starter project prepared for deployment to Vercel.
It includes example pages and a simple structure to add features:
- Text-to-Speech
- Text-to-Image
- Image-to-Text (OCR)
- Image-to-Video

## How to use
1. Extract the ZIP
2. `cd ai_multitools_starter`
3. `npm install` (or `pnpm install` / `yarn`)
4. `npm run dev` to run locally
5. Deploy to Vercel and link your GitHub repo
